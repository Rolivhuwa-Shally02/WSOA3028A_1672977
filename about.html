uses of Metadata - what, why and how?


April 1, 2021 by Mandaha RS Metadata- we can start by breaking down the word, where Meta means basic description and data is a description of something, units of information, things known as facts. Metadata is a type of data that provides information about another data, it makes searching and finding any data effective because it summarises basic information about the data. The use of metadata is useful and important when searching on the web for something for instances, searching for a book by typing the authorâ€™s name to locate it, just when you begin to search something on the web, the search engine displays a collection of data and that is what we call metadata, it can be created manually or can be automatic. Metadata is used for common things like images, videos, book, web pages etc. so basically, documents in general. There are examples of metadata that can be stored in a document namely, filename, author, date, time, file size, camera name or phone name and many more. An image comes with its metadata, the image is the data and the information on its storage is the metadata. A book as a data also has its own metadata like, the book title, author name, publisher, year it was published, page numbers, information in the beginning and back of the book. Emails, blog posts, spreadsheet and other computer files have their own metadata. There are also types of metadata which are, Administrative (manage information resources), Descriptive (describe information resources), Preservation (preservation management of resource information), Technical and use. Metadata saves/collect and preserve data so that it can be used later, it keeps records of data for a long time making it easy to access it and re-use it. It is also essential in giving location of documents, recovering files or documents. It is also useful in keeping similar searches of links or resources, it organises links and display the depending on what the user is searching. Metadata builds pages that users visit and store them in database, more like search history, it keeps resources so that they are accessible in the future, therefore it saves our life with the use of internet, and technology devices like phones where we store our documents and many files.
